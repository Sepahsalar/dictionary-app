# Dictionary App (React + TypeScript)

Live demo: https://ali-dictionary.netlify.app/

A small but polished dictionary web app built with React + TypeScript.  
Focus: clean UX, consistent UI, and production-style state handling.

## Features
- Search on submit (no noisy auto-search)
- Light/Dark theme toggle (saved in localStorage)
- Loading + error states
- Recent searches (localStorage)
- Pronunciation audio (with fallback across entries when available)
- Source link (fallback to Wiktionary)

## Tech
- React + TypeScript
- Vite
- Tailwind CSS
- lucide-react icons
- Dictionary API: https://dictionaryapi.dev/

## Run locally

npm install
